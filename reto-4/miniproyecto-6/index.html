Volver a los miniproyectos

Esta aplicación está hecha con el framework Flask. Para poder verla:

  1. Tener Python instalado en el equipo
  2. Descargar la carpeta miniproyecto-6
  3. Abrir la consola y moverse al directorio donde hayas guardado el archivo
  4. Ejecutar python app.py
